## About The Project

This app is a quick lookup user directory built with React. Searching filters users by checking all fields including name, location, and email address. Users can also be sorted alphabetically by last name.



### Built With

<ul>
	<li>React</li>
	<li>RandomUserAPI</li>
	<li>Bootstrap</li>
	<li>Github Pages</li>
</ul>






<!-- GETTING STARTED -->
### Installation

The app runs in the browser, and can be found at the following link:

[https://sprengerv.github.io/quick_lookup/](https://sprengerv.github.io/quick_lookup/)



<!-- USAGE EXAMPLES -->
## Usage

This app is for anyone looking to search a user list. It could easily be adapted to work with any database that is tied to an API that returns all users and their relevant data.
